Presto Community

User Group

Our main forum for community interaction is the presto-users group. Use this group for general questions and discussion around Presto. Join the group or send a message now.

GitHub

If you find an issue with Presto, please file an issue.

You can also contribute fixes and new features as pull requests. You will need to sign a Contributor License Agreement ("CLA") before it can be accepted into the repository.

IRC

The Presto community and developers are always active in #prestodb on Freenode. If you have any urgent questions or issues, jump in and ask the team there.

Stack Overflow

Stack Overflow is a popular place to ask questions. Read through questions tagged with prestodb or ask your own! You can also search Stack Overflow for prestodb or "Facebook Presto".

Twitter

Search for the #prestodb hashtag on Twitter to keep up with the latest Presto news.

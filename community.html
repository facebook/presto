Community Group

Our main forum for community interaction is the presto-users group. Use this group for general questions and discussion around Presto. Send a message now.

GitHub

If you find an issue with Presto, raise it as a project issue.

You can also contribute fixes and new features as pull requests. You will need to sign a Contributor License Agreement ("CLA") before it can be accepted into the repository.

IRC

The Presto community and developers are always active in #prestodb on Freenode. If you have any urgent questions or issues, jump in and ask the team there.

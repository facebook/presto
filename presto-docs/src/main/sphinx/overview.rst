************
Overview
************

Presto is a distributed query engine designed to query large datasets
distributed over several heterogenous data sources.  Presto is the
successor to a Hive designed to not only understand SQL, but to execute
SQL in a distributed query plan across a cluster of Presto
coordinators and Presto workers.
